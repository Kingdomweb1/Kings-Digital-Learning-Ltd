# Kings Digital Learning Website

This is a modern, responsive landing page for **Kings Digital Learning Ltd**, built with **HTML**, **CSS**, and **JavaScript**.

## ✅ Project structure

```
Kings Digital Learning/
├── Index.html
├── about.html
├── services.html
├── contact.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── README.md
```

## 🚀 Run locally (quick)

### Option A — Open in browser
1. Open `Index.html` in your browser (double-click or right-click → Open with).

### Option B — Run a local server (recommended)
1. Open a terminal in the `Kings Digital Learning` folder.
2. Run:

```powershell
python -m http.server 8000
```

3. Visit http://localhost:8000 in your browser.

## 🧩 How it works

- `Index.html` is the home page (hero + overview)
- `about.html` is the about/portfolio page
- `services.html` shows services and pricing
- `contact.html` includes the form and contact details
- `css/style.css` contains all layout, responsive styling, and animations.
- `js/main.js` adds:
  - mobile menu toggle
  - scroll reveal animations
  - testimonial slider
  - smooth scroll behavior

## ✨ Customization

- Replace placeholder images by updating the `src` attributes in `Index.html`.
- Update the contact email/phone/WhatsApp links in the contact section.
- Adjust colors and typography in `css/style.css` via CSS variables at the top.
